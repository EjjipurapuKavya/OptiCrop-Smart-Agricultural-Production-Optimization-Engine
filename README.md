# 🌾 OptiCrop: Smart Agricultural Production Optimization Engine

## Abstract
OptiCrop is a Machine Learning based crop recommendation system that helps farmers identify the most suitable crop based on soil nutrients and environmental conditions. The system predicts the best crop using Machine Learning algorithms and provides recommendations through a Flask web application.

---

## Features

- Crop Recommendation
- Soil Analysis
- Machine Learning Prediction
- Flask Web Application
- User Friendly Interface

---

- Python

## Dataset

The project uses the Crop Recommendation Dataset containing:

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall
- Crop Label

---

## Installation

```bash
pip install -r requirements.txt
python app.py
```

---

## How to Run

1. Install all required libraries.
2. Run `app.py`.
3. Open the browser.
4. Enter soil and environmental values.
5. Click **Predict Crop**.
6. View the recommended crop.
