# Data Collection and Analysis

## Dataset

The Crop Recommendation Dataset contains soil and environmental parameters used to recommend the most suitable crop.

## Features

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

## Data Analysis

The dataset is analyzed to understand the relationship between soil nutrients and crop suitability. Statistical analysis and visualization help identify patterns before training the machine learning model.
