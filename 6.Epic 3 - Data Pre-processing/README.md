# Data Pre-processing

## Overview

Data preprocessing is an important step in machine learning. It improves the quality of the dataset before model training.

## Steps Performed

- Data Cleaning
- Handling Missing Values
- Removing Duplicate Records
- Feature Selection
- Feature Scaling
- Train-Test Split

## Tools Used

- Python
- Pandas
- NumPy
- Scikit-learn

## Outcome

The processed dataset is ready for training the machine learning model.
